## About

저는 웹, 앱, 결제, 데이터 파이프라인, 인프라, 서비스 운영까지 경험한 Senior Software Engineer / Tech Lead입니다.

2006년부터 소프트웨어 엔지니어링의 전 주기를 경험했고, 최근 5년간은 글로벌 한국어 교육 플랫폼 Talk To Me In Korean에서 엔지니어링 팀장으로 일하며 결제 시스템, 데이터 파이프라인, 앱/웹 서비스, 레거시 인프라 정리, 운영 자동화 업무를 주도했습니다.

저의 강점은 단순히 코드를 작성하는 것을 넘어, 기술·운영·비즈니스 사이를 연결해 실제 운영 가능한 구조로 문제를 해결하는 것입니다.
